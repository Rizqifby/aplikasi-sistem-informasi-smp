# SPEBRA - Aplikasi Sistem Informasi Sekolah

Aplikasi Android ini dibuat untuk sistem informasi digital SMP Negeri 1 Brangsong. 
Proyek ini dikerjakan secara berkelompok sebagai tugas akhir kelulusan jurusan RPL di SMK Negeri 4 Kendal (Agustus 2022 - Februari 2023).

Aplikasi ini dibangun menggunakan:
- Android Studio
- Java 

Tugas dan peran saya dalam proyek ini:
- Programmer: Mengurus bagian coding dan logika aplikasi menggunakan Java.
- UI/UX Designer: Membuat desain antarmuka aplikasi agar rapi dan mudah dipahami.
